# Dulce Aroma

## Descripción

Aplicación desarrollada en Java que permite administrar los productos de una cafetería mediante operaciones CRUD (Crear, Consultar, Actualizar y Eliminar), utilizando conexión a una base de datos MySQL a través de JDBC.

## Tecnologías utilizadas

- Java
- Apache NetBeans
- MySQL (XAMPP)
- JDBC
- Git
- GitHub

## Funcionalidades

- Registrar productos.
- Consultar productos.
- Actualizar productos.
- Eliminar productos.
